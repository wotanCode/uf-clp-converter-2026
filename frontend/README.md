# Conversor UF/CLP FRONTEND

Aplicación web para convertir montos entre UF y Pesos Chilenos (CLP) consumiendo la API del backend.

## Tecnologías

Para el desarrollo de este proyecto se utilizaron las siguientes herramientas:

- **Vue 3**: Framework para la construcción de interfaces de usuario.
- **Tailwind CSS**: Framework CSS para un diseño limpio y responsivo.
- **Vue Router**: Enrutador oficial para la navegación entre las vistas de la aplicación.
- **Pinia**: Gestor de estado para el manejo centralizado de datos.
- **Axios**: Cliente HTTP para realizar las peticiones a la API del backend.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Aplicación disponible en `http://localhost:5173`
